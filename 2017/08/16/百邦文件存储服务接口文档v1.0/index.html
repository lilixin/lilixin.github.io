百邦文件存储服务接口文档v1.0

接口签名

本文档中所有接口均需要签名才可正常调用

签名说明

由服务提供者提供统一的secretKey

签名计算

拼接userId、secretKey并求MD5值

参考

例如：
secret=9AC75679E9881E6B
userId=123456
计算签名 sign=MD5(secret+ userId)
既sign=MD5(9AC75679E9881E6B123456).toUpperCase()

最后提交请求时把sign当做参数提交
